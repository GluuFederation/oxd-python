set_expression(http_method, expression)[source]¶

Set a scope expression scope_expression is Gluu invented extension which allows to put JsonLogic expression instead of single list of scopes. Please read more about scope_expression at https://gluu.org/docs/ce/3.1.2/admin-guide/uma.md

Parameters:

  • http_method (str): a HTTP method like GET, POST, PUT, DELETE

  • expression (dict): the scope expression in the format:

    {
  "rule": {
    "and": [
      {
        "or": [{"var": 0}, {"var": 1}]
      },
      {
        "var": 2
      }
    ]
  },
  "data": [
    "http://photoz.example.com/dev/actions/all",
    "http://photoz.example.com/dev/actions/add",
    "http://photoz.example.com/dev/actions/internalClient"
  ]
}
